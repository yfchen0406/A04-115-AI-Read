讀書心得圖片 Prompt 與配色示範

將一段文字轉成左圖 / 右圖的具體提示詞，並搭配最適合中圖風格的色盤與字體尺寸。

一、示範文字

美國巧妙地把「想賺錢」包裝成「天命所歸」，用宗教理由合理化對財富的追求，為資本主義的瘋狂發展打下堅實的心理基礎。

這段內容適合拆成兩條線：左圖做邏輯與重點整理，右圖做情緒與象徵延伸，中間主圖則繼續保留你目前最滿意的故事型畫面。

左圖｜資訊型 Prompt
角色任務：請你擔任教育型資訊圖設計師，把一段歷史觀點整理成清楚易讀的摘要圖卡。 背景資訊：主題是「信仰與商業的共生」，內容強調宗教信念如何讓財富追求被視為正當，進一步形成資本主義發展的心理基礎。 具體指令：製作一張橫式資訊圖，畫面分成 3 到 4 個模組，依序呈現「神選觀念」、「賺錢即祝福」、「適者生存」、「資本擴張」；每個模組搭配簡單圖示、箭頭流程、少量中文字短標；版面乾淨、重點明確、留白清楚。 約束條件：採用扁平插畫加簡報式排版，色調使用米白、淺藍灰、暖金棕，中文字清楚可讀，整體像高質感讀書筆記圖卡，16:9 橫式構圖。
右圖｜情緒型 Prompt
角色任務：請你擔任概念插畫設計師，把一段抽象的歷史評論轉成有情境感與象徵性的畫面。 背景資訊：主題是「把對財富的追求包裝成神聖使命」，重點在於宗教與商業結合後，讓人把逐利視為合理、甚至高尚的行為。 具體指令：生成一張具有故事感的概念插畫，畫面中可出現教堂、金幣、商人、光環、工廠或象徵資本擴張的城市元素；整體呈現「財富被神聖化」的視覺隱喻；右下角預留一句短標語的位置。 約束條件：使用敘事插畫風、略帶復古教育海報感，色調與中間主圖一致，主色為米金棕、霧藍灰、柔和暖光，畫面細節中等，不要過度擁擠，16:9 橫式構圖。

三、字體尺寸示範

大標題：36–44 px
小標題 / 模組標題：22–28 px
內文短句：16–20 px，適合左圖模組說明與右圖一句結論。
補充小字：13–15 px，適合角標、來源、提示語。

如果中圖本身元素較多，左右圖的文字請再減少，尤其右圖保留一句 10 到 14 字的短標語最穩定。
