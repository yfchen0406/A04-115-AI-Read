完整功能＆指令白話教室

以下每一個「斜槓指令」都是讓Claude更專業的鑰匙，搭配使用效果驚人。

/brainstorming 🎯 當你需要： refine an idea
💡 打磨點子・瘋狂腦暴

白話解釋： 你有一個模糊的想法或粗略概念，輸入 /brainstorming 後，Claude 會像專業創新顧問一樣，幫你延伸、發散、提煉出不同角度與可能性，甚至找出盲點。

✨ 實際用法：貼上你的點子草稿，加上「/brainstorming」，Claude 會產出 10 種延伸方向、潛在風險和執行路徑。
/skill-creator 📚 當你需要： create new skills
🛠️ 創造新技能｜自訂學習路徑

白話解釋： 不只問問題，還能讓Claude幫你「設計一套新能力的養成計畫」。無論是想學會某項軟體、溝通技巧，或是提升某種專業，它會拆解學習步驟並提供練習框架。

🎯 舉例：輸入「/skill-creator 我想培養數據分析思維」，Claude 會產出從入門到實戰的完整學習地圖與資源建議。
/writing-plans 📋 當你需要： write plans
📌 寫出縝密計畫書

白話解釋： 專案計畫、行銷排程、個人年度目標…… 只要用這個指令，Claude 會用結構化的方式為你產出「目標、資源、時間軸、風險評估」的完整計畫。

📝 實際應用：輸入「/writing-plans 三個月內推出個人網站」，Claude 會自動產出甘特圖風格文字計畫＋每週里程碑。
/executing-plans ⚙️ 當你需要： execute tasks
🚀 執行力模式｜把計畫變行動

白話解釋： 有了計畫但不知如何下手？這個指令讓Claude化身專案執行教練，將大目標切分成具體、可立刻做的子任務，並提供清單、工具建議與追蹤方法。

🔧 舉例：上傳你的計畫摘要 +「/executing-plans」，Claude 會產出第一週每日行動清單，甚至提醒常見卡關點。
/ui-ux-pro-max 🎨 當你需要： design UI components
🎨 頂級 UI/UX 元件設計

白話解釋： 專門產生現代化、高品質的前端元件（按鈕、表單、卡片、導航列等），附帶 CSS / Tailwind 程式碼及無障礙設計建議。告別普通「AI 雜亂風格」。

💎 使用技巧：輸入「/ui-ux-pro-max 登入表單 + 深色模式」，Claude 會回傳可直接複製的 HTML/CSS 元件，且符合專業設計規範。
/frontend-design 🚫 當你需要： avoid AI slop design
✨ 擺脫廉價AI風格｜高質感前端

白話解釋： 許多 AI 生成的版面都長得很「模板感」、缺乏細節。這個指令會讓 Claude 產生精緻、有層次、具有現代感的前端程式碼，乾淨且高度客製化。

🌟 實際用法：下達「/frontend-design 製作一個復古未來風的定價卡片」，輸出的設計將脫離典型AI濫觴，更有設計師品味。
/brave search 🌐 當你需要： access live internet
🔍 Brave 即時搜尋引擎

白話解釋： 讓 Claude 能夠即時上網抓取最新資料，而不是只依賴過時的訓練數據。透過 Brave Search API，取得新聞、產品價格、即時資訊，並附上來源。

⚡ 例子：輸入「/brave search 2026 年最佳AI工具排名」，Claude 會提供當前網路上的真實清單與評測連結。
/socialcrawl 📱 當你需要： live social media data
📊 社群媒體即時數據爬取

白話解釋： 分析當下流行的社群貼文、熱門話題、競爭者動態。能抓取 Twitter / Reddit / Threads 等平台的公開數據摘要，讓你即時掌握風向。

📈 案例：輸入「/socialcrawl 最近 24 小時關於 AI 繪圖的熱門討論」，Claude 會回饋趨勢總結與代表性貼文觀點。
/firecrawl 🔥 當你需要： live web data
🔥 Firecrawl 深度網頁資料

白話解釋： 比一般搜尋更深入，能夠爬取特定網站的結構化內容（例如產品規格、新聞內文、價格表），並將多個頁面統整成易讀摘要。

🕸️ 實際應用：輸入「/firecrawl 官網 最新優惠活動＋規格表」，Claude 會幫你抓取並整理成比較表格，省去手動逐頁查看的時間。
/feature-dev ⚙️ 當你需要： think like engineer
🧑‍💻 工程師思維｜功能開發模式

白話解釋： 讓Claude模擬頂尖軟體工程師的邏輯，針對功能需求進行系統分析、架構設計、演算法建議，並產出可執行的技術方案或程式碼片段。

🛠️ 例子：輸入「/feature-dev 開發一個即時聊天室的後端 API」，Claude 會提供 Node.js + WebSocket 的設計思路、資料庫結構及安全性考量。
