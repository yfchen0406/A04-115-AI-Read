撰寫與寄信
撰寫新郵件
點擊「撰寫」建立新郵件，支援富文字格式、附件與多位收件人。
Prompt 範例
幫我撰寫一封正式中文商業邀請函，邀請廠商參加下個月產品發表會，語氣專業友善，200 字以內。
回覆與轉寄
快速回覆單一寄件人或全部收件人，或將郵件轉寄給第三方。
Prompt 範例
對方詢問報價，幫我婉拒並保留未來合作機會，語氣客氣，3 句話以內。
附件與插入圖片
可附加本機檔案、雲端硬碟檔案，或直接在郵件內嵌入圖片。
Prompt 範例
我要寄含報表附件的 Email，幫我寫一段附件說明，提醒對方為 PDF 格式需 Acrobat 開啟，30 字以內。
排程傳送
設定未來特定日期時間自動寄出郵件，避免深夜打擾收件人。
Prompt 範例
我想週一早上 9 點準時發出活動通知，請給我 Gmail「排程傳送」的操作步驟，5 步以內。
收回已傳送郵件
啟用「傳送復原」，可在寄出後 30 秒內撤回郵件，避免寄錯。
Prompt 範例
說明 Gmail「傳送復原」的設定位置與時間限制，以及寄錯信後的補救方法。
群組 / 通訊錄寄信
使用 Google 通訊錄建立聯絡人群組，一次寄送給多位成員。
Prompt 範例
我有 20 位讀書會成員的 Email，告訴我如何在 Google 通訊錄建立群組，一鍵群發到 Gmail。
整理與收件管理
標籤 (Labels)
建立自訂標籤，如同資料夾，將郵件分類歸檔，支援多重標籤。
Prompt 範例
我是退休教師，請建議適合個人學習管理的 Gmail 標籤架構，涵蓋 AI、閱讀、英文、生活帳單，5 個以內。
星號標記與多色星號
用星號標記重要郵件，進階設定可啟用不同顏色與形狀的星號。
Prompt 範例
說明 Gmail 多色星號的啟用方式，並建議用顏色區分「緊急」「待辦」「參考」三種狀態的方法。
封存與刪除
封存讓郵件從收件匣消失但保留搜尋；刪除則移至垃圾桶 30 天後清除。
Prompt 範例
比較 Gmail 封存與刪除的差異，建議哪類郵件適合封存、哪類直接刪除，用表格呈現。
收件匣類型設定
可選擇預設、優先、多重收件匣等版面，自動分類主要、社交、促銷郵件。
Prompt 範例
解釋 Gmail 各收件匣類型的差異，哪種最適合想專注重要信件的退休人士？
對話串 (Threading)
同一主旨往返郵件自動聚合成對話串，方便追蹤完整脈絡。
Prompt 範例
說明如何關閉 Gmail 對話串功能，改為每封郵件獨立顯示，並比較各自的優缺點。
提醒稍後處理 (Snooze)
對郵件設定提醒，讓郵件在指定時間重回收件匣頂端。
Prompt 範例
告訴我 Gmail Snooze（稍後提醒）功能的操作方式，以及如何為需週五前回覆的信設定提醒。

進階搜尋運算子

依日期範圍搜尋
使用 before: after: older_than: 等語法鎖定特定時段的郵件。
Prompt 範例

封鎖寄件人

Prompt 範例

儲存搜尋 / 快捷搜尋

自動化與設定
度假自動回覆
設定外出自動回覆，對所有或指定聯絡人自動發出通知信。
Prompt 範例

罐頭回覆（範本）

AI 智慧功能 (Gemini)
Gemini 協助撰寫
撰寫郵件時呼叫 Gemini AI，輸入大意即可生成完整草稿並調整語氣。
Prompt 範例
使用 Gmail Gemini 幫我把「想向舊同事問候，順便說說退休後的生活」擴展為一封溫暖的中文信件。
郵件智慧摘要
Gemini 自動摘要長串郵件，讓你快速掌握重點，不必逐封閱讀。
Prompt 範例
這封郵件串有 15 封往返，請用 Gemini 幫我摘要出：主要決議、待辦清單、下次會議時間。
智慧回覆建議
Gmail 分析郵件內容，在回覆欄下方提供 3 個快速回覆選項。
Prompt 範例
說明 Gmail「智慧回覆」功能的運作原理、如何讓它更符合個人語氣偏好，以及使用限制。
智慧撰寫 (預測輸入)
輸入文字時 Gmail 自動補全句子，按 Tab 接受建議加速撰寫。
Prompt 範例

郵件翻譯功能
收到外語郵件時 Gmail 自動偵測並提供翻譯，支援 100 多種語言。
Prompt 範例
