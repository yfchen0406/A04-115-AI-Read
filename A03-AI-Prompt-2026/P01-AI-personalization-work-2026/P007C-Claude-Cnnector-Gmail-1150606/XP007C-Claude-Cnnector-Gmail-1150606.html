學習可能性
撰寫與寄信
撰寫新郵件
點擊「撰寫」建立新郵件，支援富文字格式、附件、收件人欄位。
Prompt 範例
幫我撰寫一封正式的中文商業邀請函，邀請廠商參加下個月的產品發表會，語氣專業但友善，200 字以內。
回覆與轉寄
快速回覆單一寄件人或全部收件人，或將郵件轉寄給第三方。
Prompt 範例
對方詢問報價，請幫我婉拒並說明目前暫不接新案，但保留未來合作機會，語氣客氣，3 句話以內。
附件與插入圖片
可附加本機檔案、雲端硬碟檔案，或直接在郵件內嵌入圖片。
Prompt 範例
我要寄出含報表附件的 Email，幫我寫一段附件說明文字，提醒對方報表為 PDF 格式，需 Acrobat 開啟，30 字以內。
排程傳送
設定未來特定日期時間自動寄出郵件，避免深夜打擾收件人。
Prompt 範例
我想在週一早上 9 點準時發出活動通知，請給我 Gmail「排程傳送」的操作步驟，5 步以內。
收回已傳送郵件
啟用「傳送復原」功能，可在寄出後 30 秒內撤回郵件。
Prompt 範例
解釋 Gmail「傳送復原」功能的設定位置與限制，以及我不小心寄錯信後的補救方式。
群組 / 通訊錄寄信
使用 Google 通訊錄建立聯絡人群組，一次寄送給多位成員。
Prompt 範例
我有 20 位讀書會成員的 Email，告訴我如何在 Google 通訊錄建立群組，以後在 Gmail 一鍵群發。
整理與收件管理
標籤 (Labels)
建立自訂標籤，如同資料夾，將郵件分類歸檔，支援多重標籤。
Prompt 範例
我是退休教師，請建議一套適合個人學習管理的 Gmail 標籤架構，涵蓋 AI、閱讀、英文、生活帳單，5 個以內。
星號標記與多色星號
用星號快速標記重要郵件，進階設定可啟用不同顏色與形狀的星號。
Prompt 範例
說明 Gmail 多色星號功能的啟用方式，並建議一套用顏色區分「緊急」、「待辦」、「參考」的方法。
封存與刪除
封存讓郵件從收件匣消失但保留搜尋，刪除則移至垃圾桶 30 天後清除。
Prompt 範例
比較 Gmail「封存」與「刪除」的差異，並建議哪類郵件適合封存、哪類直接刪除，用表格呈現。
收件匣類型設定
可選擇預設、優先、多重收件匣等版面，自動分類主要、社交、促銷郵件。
Prompt 範例
解釋 Gmail 收件匣各類型（預設、優先、多重收件匣）的差異，哪種最適合想專注重要信件的退休人士？
對話串 (Threading)
同一主旨的往返郵件自動聚合成對話串，方便追蹤完整脈絡。
Prompt 範例
說明如何關閉 Gmail 對話串功能，改為每封郵件獨立顯示，並說明各自的優缺點。
提醒稍後處理
對郵件設定「提醒」，讓郵件在指定時間重回收件匣頂端提醒你。
Prompt 範例
告訴我 Gmail Snooze（稍後提醒）功能的操作方式，以及如何為一封需要週五前回覆的信設定提醒。
搜尋與篩選
進階搜尋運算子
使用 from: to: subject: has:attachment 等指令精確定位郵件。
Prompt 範例
列出最常用的 10 個 Gmail 搜尋運算子，每個附上實際範例，以表格呈現，包含中文說明。
篩選器建立
設定條件自動對特定來源郵件套用標籤、封存、標記等動作。
Prompt 範例
幫我設計一個 Gmail 篩選器，讓所有來自電商促銷的郵件自動加上「促銷」標籤並跳過收件匣，操作步驟說明。
依日期範圍搜尋
使用 before: after: older_than: 等語法鎖定特定時段的郵件。
Prompt 範例
我想找出 2024 年全年收到的含附件郵件，請給我完整的 Gmail 搜尋語法指令。
封鎖寄件人
封鎖特定寄件人，之後來信自動移至垃圾郵件，可隨時解除。
Prompt 範例
說明在 Gmail 封鎖特定寄件人的步驟，以及封鎖後郵件的去向，和封鎖與篩選的差別。
儲存搜尋條件
將常用搜尋條件儲存為篩選器，不必每次重新輸入複雜語法。
Prompt 範例
我每週都要查閱含「發票」字樣的郵件，如何在 Gmail 建立一個可重複使用的搜尋快捷方式？
自動化與設定
自動回覆 (度假回覆)
設定外出自動回覆，對所有或指定聯絡人自動發出通知信。
Prompt 範例
幫我寫一段 Gmail 度假自動回覆的英文範本，說明我休假到月底，緊急聯絡請打電話，語氣友善專業。
罐頭回覆 (範本)
預先儲存常用回覆範本，需要時一鍵插入，節省重複打字時間。
Prompt 範例
幫我設計 3 種 Gmail 罐頭回覆範本：①確認收到 ②需要更多時間 ③婉拒請求，各 50 字以內。
多個寄件身分
新增其他 Email 地址作為寄件身分，統一用 Gmail 管理多個信箱。
Prompt 範例
說明如何在 Gmail 設定以不同寄件人身分（如個人信箱與部落格信箱）寄信，並設定各自的預設簽名。
簽名檔設定
建立含格式、連結、圖片的 HTML 簽名檔，可為不同身分設定不同簽名。
Prompt 範例
幫我撰寫一段 Gmail 簽名檔文字，包含：退休教師、AI 學苑部落格連結、聯絡 Email，格式簡潔。
鍵盤快捷鍵
啟用快捷鍵後，可用 c 撰寫、r 回覆、j/k 移動等鍵操作，大幅提升效率。
Prompt 範例
列出 Gmail 最實用的 10 個鍵盤快捷鍵，以表格呈現，包含「鍵位」「功能說明」「適合情境」三欄。
匯入其他信箱
透過 POP3 設定將其他信箱郵件匯入 Gmail 統一管理。
Prompt 範例
說明如何將 Outlook 或 Yahoo 信箱透過 POP3 匯入 Gmail，並列出可能遇到的常見問題與解法。
安全性與隱私
機密模式
設定郵件到期日與 SMS 密碼驗證，防止收件人轉寄或複製內容。
Prompt 範例
說明 Gmail 機密模式的功能與限制，以及哪些情況下適合使用，並比較與一般加密的差異。
兩步驟驗證
開啟 Google 帳號兩步驟驗證，即使密碼外洩也能保護 Gmail 安全。
Prompt 範例
為什麼 Gmail 一定要開啟兩步驟驗證？說明開啟步驟、驗證方式選項，以及備用碼的重要性。
垃圾郵件管理
Gmail 自動過濾垃圾郵件，可手動標記、解除標記，訓練過濾效果。
Prompt 範例
我的 Gmail 常收到假冒銀行的釣魚信，說明如何辨識釣魚郵件的特徵，並設定讓 Gmail 更準確過濾。
帳號活動查看
查看最近登入記錄與裝置，發現異常即可立即登出所有裝置。
Prompt 範例
說明如何在 Gmail 查看帳號最近的登入活動，判斷是否有異常，以及發現可疑登入後的緊急處理步驟。
取消訂閱功能
Gmail 在促銷信件上方顯示「取消訂閱」按鈕，一鍵退訂電子報。
Prompt 範例
我收件匣塞滿電子報，給我一個有效清理 Gmail 促銷訂閱的完整流程，包含批次取消訂閱的方法。
AI 智慧功能 (Gemini)
Gemini 協助撰寫
在撰寫郵件時呼叫 Gemini AI，輸入大意即可生成完整草稿並調整語氣。
Prompt 範例
在 Gmail 使用 Gemini 協助寫信功能，幫我把「想向舊同事問候近況，順便說說自己退休後的生活」擴展為一封溫暖的中文信件。
郵件智慧摘要
Gemini 自動摘要長串郵件，顯示重點摘要讓你快速掌握內容。
Prompt 範例
這封郵件串有 15 封往返信件，請用 Gemini 功能幫我摘要出：主要決議事項、待辦清單、下次會議時間。
智慧回覆建議
Gmail 自動分析郵件內容，在回覆欄下方提供 3 個快速回覆選項。
Prompt 範例
說明 Gmail「智慧回覆」功能的運作原理，以及如何讓它更符合個人語氣偏好，有哪些限制？
智慧撰寫建議
輸入文字時 Gmail 自動補全句子，按 Tab 接受建議，加速撰寫。
Prompt 範例
Gmail 的「智慧撰寫」預測輸入功能如何運作？如何開啟或關閉，對英文寫作的幫助是否比中文大？
郵件翻譯功能
收到外語郵件時 Gmail 自動偵測並提供翻譯，支援 100 多種語言。
Prompt 範例
我收到一封日文商業郵件，說明如何使用 Gmail 內建翻譯功能，以及翻譯結果的準確度限制與注意事項。
