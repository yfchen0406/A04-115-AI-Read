2

任務驅動式提問：給出明確約束，讓 AI 主動規劃。實作待辦清單網站、理解他人專案流程。

進階區 (1-2週+)

⚙️ 自動化與協作

💼 實戰案例：從零打造「待辦清單網站」

使用 任務驅動式提問，讓 Codex 自動生成完整 Flask 應用，包含資料庫與前端。

「幫我做一個待辦清單網站：
- 使用 Flask 框架
- 需要有新增、完成、刪除待辦事項的功能
- 資料存在 SQLite 資料庫裡
- 完成後幫我啟動它」
成果： Codex 將自動產生 app.py、templates/index.html，建立資料庫模型，並提供即時預覽網址。整個過程只需 30 秒～2 分鐘。
案例二：理解開源專案驗證流程

當你拿到一個陌生程式碼倉庫，在 Codex 對話框輸入：

「這個專案的身份驗證功能是怎麼實現的？請幫我梳理整個流程。」

Codex 會掃描相關檔案，以流程圖文字描述 token 驗證、session 管理或 OAuth 步驟，幫你快速上手。

✨ 提問心法：從模糊到精準

低效問法

「幫我寫個網站」
「這個函數怎麼改」

高效提問 (任務驅動)

「用 React + Tailwind 做一個記帳儀表板，需要圖表和支出分類，並儲存到 localStorage」
→ 提供框架、功能邊界、儲存方式。
