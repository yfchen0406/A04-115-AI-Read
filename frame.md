# 第三部分~筆記當中可以規劃為space部分

- **B01-製作封面時**
    - 說明~製作封面時使用，[ ]中的人物可以替換
    - 指令
        - 下面資料做一張封面，你幫我產出一段給Nano Banana pro 的資訊圖表提示詞，格式4:3，並且我要[下方第一行]人物當配角，字體以鋼筆字體為主，標題放在中間。
- **B02-製作網頁橫幅**
    - 說明~網頁橫幅為26:9，一次產出上下二部分切割使用
    - 指令
        - 幫我產生一張橫幅圖片的提示詞，要放在NanoBanana pro來產出圖片，比例26:9，區分上下二部分，上半部以[下方第1行]人物為主，下半部以[下方第2行]人物為主，圖片內容呈現要能夠配合下面這句話，上下半部都以手繪插畫風格為主，各自獨立可以裁切開來，當然下面這句話也要各自獨立出現，並且一定要是繁體中文字樣，字體以鋼筆字體為主，標題放在上方
- **B03-製作英文單字圖卡**
    - 說明~這一部分製作英文單字卡時使用，還可以修改
    - 指令
        - 配合以下英文單字(標註單字詞類如v.或n.片語等)約4-6個單字，以手繪插畫風格呈現一張英文單字故事圖卡，以[下方第1行]人物為配角，內容以配合英文單字內容為主，比例為16:9，並且中文必須以繁體中文呈現，字體以鋼筆字體為主，標題放在上方，幫我產生一套統一規格的單字圖卡提示詞，適用Nano Banana pro 使用，並且配角人物可以代換
