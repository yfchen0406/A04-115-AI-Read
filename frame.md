# 四十一：退伍兵抗議拖欠款　羅斯福危機下登台

- 這段講的是美國在1929年大蕭條最慘的時候，爆發了「補助金禁軍」老兵抗議事件。
- 這場悲劇讓胡佛總統聲望崩盤，也讓二十世紀最重要的政治人物羅斯福趁勢崛起，開啟了史無前例的四屆總統任期，徹底改變了美國政黨格局和政府該扮演什麼角色。

## 1. 老兵快餓死跑去跟政府討債

- 1929年大蕭條讓美國四分之一的人失業，五千多家銀行倒閉，很多人連飯都吃不上。
- 一群一戰退伍老兵實在活不下去，集結兩萬多人跑到華盛頓，要求政府提前發放原本要等到1945年才給的補助金。
- 沒想到胡佛總統不但拿不出錢，還怕別人學樣，最後竟派麥克阿瑟和巴頓帶領正規軍隊，用武力鎮壓這些曾經為國打仗的老兵。
- 這場流血衝突讓胡佛被罵成冷血無能，也給了對手羅斯福絕佳的攻擊機會。

## 2. 最強男主角打破紀錄連任四次

- 羅斯福是美國二十世紀最重要的人物，地位可比華盛頓和林肯。
- 他是唯一當過四屆總統的人，主要是因為當時大蕭條加上二戰，民眾怕換人會讓政策中斷。
- 不過美國人也很怕出現獨裁者，所以羅斯福一死，馬上修憲規定總統最多只能做兩任——寧可犧牲讓偉人多幹幾年的好處，也要確保國家不會走向專制。

## 3. 權力大到沒人攔得住的變革

- 羅斯福在位時，原本互相牽制的三權分立幾乎失效，他的個人威望高到國會和最高法院都不敢正面對抗。
- 靠著這種「說了算」的地位，他推動非常激進的「新政」，把民主黨從原本主張「政府少插手」，轉變成「政府要大力救經濟」。
- 這個一百八十度的大轉彎，不只讓美國政黨的支持群體大洗牌，也讓後來的政治鬥爭，變成「支持新政」vs.「反對新政」的長期對立。

# 四十二：羅斯福新官三把火　參眾院跪舔無極限

- 這段講的是美國史上影響力最大的總統之一富蘭克林·羅斯福，如何在大蕭條亂局中崛起。
- 內容包括他顯赫卻有爭議的私生活、隱藏殘疾的政治手腕，以及上台後用「新政」擴大政府權力、試圖拯救經濟，最終重塑美國政治生態。

## 1. 大咖也有私生活：隱藏疾病的政治巨人

- 羅斯福出身名門，從小接受頂尖教育。
- 他雖然很會交際，但私生活有污點——曾背叛內向的妻子，和她的秘書外遇，後來為了政治前途才勉強維持婚姻。
- 四十歲左右他得了小兒麻痺，雙腿癱瘓，但為了選舉形象，他用金屬支架撐著走路，在公開場合絕不讓人看到自己坐輪椅。
- 這種堅強又精明的做法，讓他成功在那個重視領袖體格的年代脫穎而出。

## 2. 趁著對手太爛，羅斯福順勢上位

- 1932年大選時，胡佛政府對大蕭條完全沒辦法，民怨爆棚，甚至有人說「民主黨派條狗都能贏」。
- 羅斯福在肯尼迪總統老爸（老約瑟夫·甘迺迪）的幫忙下出線，被渴望改變的民眾當成救星。
- 他上任頭一百天，趁著民意最高、國會又是同黨掌控的黃金時機，快速通過一堆法律，史稱「百日新政」。
- 這波「新官上任三把火」，後來成了所有美國總統上任後的表現標準。

## 3. 打破自由市場傳統的「政府管家」模式

- 羅斯福新政的核心是三個R：救濟（Relief）、復興（Recovery）、改革（Reform）。
- 這套做法讓政府開始大力干預經濟，加強金融監管、打擊投機，在當時其實有點像「社會主義」，挑戰了美國長期信奉的自由市場原則。
- 這是因為當時全球經濟崩盤，光靠市場自己調節已經沒用了。
- 他關閉銀行防止擠兌、讓美元脫離金本位、強迫證券公司公開財報，試圖重建金融秩序。

## 4. 撒錢搞建設，甚至叫農民把牛奶倒掉

- 羅斯福砸大錢蓋水壩、修公路、建學校，想用政府投資創造就業來刺激經濟。
- 為了拉高暴跌的農產品價格，他還推出減產政策——結果出現「農民把牛奶倒進溝裡，窮人卻喝不到」的荒謬畫面。
- 這其實是為了減少供應、抬高價格，好讓農民有收入活下去。
- 雖然新政讓國會對總統幾乎言聽計從，但美國經濟真正恢復，其實要等到二戰開打；而民主黨也在1937年中期選舉慘敗。

# 四十三：三權分立岌岌可危　戰爭陰霾籠罩全美

- 這段講羅斯福在第二、第三任期面對的兩大難題：對內差點跟最高法院鬧翻，動搖三權分立；對外則要在民間強烈「孤立主義」和備戰之間小心平衡，最後帶領美國走上世界舞台中心。

## 1. 總統跟大法官吵架，差點搞垮體制

- 羅斯福的新政很受歡迎，但最高法院的保守派大法官卻接連判他的政策違憲。
- 他氣得提出「法院改組計畫」，想多塞幾個自己人進去，讓投票結果聽他的。
- 這招雖逼得大法官妥協或退休，保住了三權分立的底線，但也讓國會嚇壞，覺得他想當獨裁者。
- 結果他在1938年中期選舉慘敗，失去國會無條件支持，新政推進也大幅放緩。

## 2. 外面亂糟糟，美國人卻只想關門過日子

- 當時全世界像個火藥桶：希特勒在歐洲擴張，日本全面侵華，二戰一觸即發。
- 羅斯福很想幫盟友，但美國民眾九成以上反對參戰，這就是「孤立主義」——大家覺得一戰被歐洲騙去送死，現在只想安心賺錢，不想孩子再上戰場。
- 羅斯福只好一邊偷偷跟邱吉爾、蔣介石聯絡，一邊對國內演戲安撫反戰情緒。

## 3. 打破傳統連任三次，全靠這句保證

- 1940年，羅斯福挑戰史無前例的第三任。
- 他能贏，除了新政威望，關鍵是他公開承諾：「只要我當總統，絕對不讓美國捲入戰爭。」
- 這句話打中孤立主義者的心，大家怕換人反而會惹禍，於是又選他。
- 雖然嘴上說不參戰，但他心裡清楚躲不掉，一上任就開始秘密擴軍備戰，以防萬一。

## 4. 用「四大自由」把全民拉進正義戰場

- 羅斯福為了慢慢改變民意，提出著名的「四大自由」：言論自由、信仰自由、免於匱乏的自由、免於恐懼的自由。
- 他想把美國塑造成「民主堡壘」，去幫助被侵略的國家。
- 他透過廣播不斷喊話，但孤立主義還是很強。
- 直到日本偷襲珍珠港，給了他一個誰都無法反對的理由，美國才正式撕掉和平假面具，全力投入二戰。

# 四十四：英法綏靖妄圖嫁禍　納粹勢大直搗巴黎

- 這段分析第二次世界大戰爆發的必然性與早期戰況，重點講德、義、日如何因困境與野心走向極端，以及英法「綏靖政策」的算計失誤，導致防線崩潰。
- 最後因德國攻蘇聯、日本炸珍珠港，戰爭升級為全球混戰。

## 1. 為什麼世界大戰非打不可？

- 二戰根本是一戰沒解決完的續集。
- 德國被《凡爾賽條約》壓得喘不過氣，又遇上大蕭條，人民苦不堪言，才讓希特勒這種極端份子上台。
- 義大利覺得分贓太少，日本想稱霸亞洲，三國各有盤算。
- 開打前他們早就「熱身」：在西班牙內戰測新武器、日本全面侵華。
- 這一切混亂，都是因為舊秩序守不住，新勢力又想用武力搶地盤，大戰幾乎不可避免。

## 2. 英法想當好人，結果被騙慘了

- 面對希特勒一步步吞併土地，英國和法國選擇「讓步」，就是「綏靖政策」。
- 背後有三個想法：一是希特勒每次都說「這是最後一次」，他們居然信了。
- 二是法國被一戰嚇怕，寧可退讓也不想再死人。
- 三是英法暗自希望德國往東打，最好跟蘇聯拼個你死我活。
- 結果希特勒和史達林私下簽互不侵犯條約，還一起瓜分波蘭，讓想坐收漁利的英法傻眼。

## 3. 法國引以為傲的防線成了笑話

- 1939年德國入侵波蘭，二戰正式開打。
- 法國自信滿滿，花大錢建了「馬奇諾防線」，以為能像一戰那樣死守。
- 沒想到德國玩「閃電戰」，直接繞過防線，從防守最弱的比利時突襲。
- 法國瞬間被打懵，巴黎淪陷、宣布投降。
- 有趣的是，義大利看德國快贏了，想來撿便宜，結果連法國殘兵都打不贏。
- 英國眼看不妙，趕緊在敦克爾克撤軍，還把自己留在法國的軍艦炸掉，免得被德軍拿走。

## 4. 英國死守到底，二戰迎來大轉折

- 法國投降後，希特勒想逼英國也投降，但首相邱吉爾硬是不低頭。
- 靠著英吉利海峽這道天險，德國陸軍過不去，空襲也沒打垮英國，戰事陷入僵局。
- 但1941年發生兩件大事：一是德國撕約攻打蘇聯，開闢巨大新戰場。
- 二是日本偷襲珍珠港，把美國正式拖下水。
- 這兩件事一發生，原本分散的戰火串成一體，二戰正式變成全球大混戰。

# 四十五：希特勒頭鐵攻蘇俄　小日本轟炸珍珠港

- 這段聚焦二戰關鍵轉折點：希特勒打蘇聯、日本炸珍珠港。
- 分析這些決策背後的算計與誤判，以及它們如何終結美國孤立主義，讓這頭「工業巨獸」參戰，徹底扭轉戰局。

## 1. 元首頭鐵去捅蘇聯這個馬蜂窩

- 1941年，希特勒做了被後世罵翻的決定：撕毀條約進攻蘇聯。
- 他以為蘇聯剛經歷大清洗，軍隊弱爆了，加上德軍在歐洲連戰連勝，覺得自己穩贏。
- 他打的是「種族優越」的主意，認為斯拉夫人是劣等民族，不配擁有土地和石油，想用閃電戰搶資源稱霸。
- 沒想到這一打，戰線拉超長，還把蘇聯逼到英國那邊，德國從此被兩面夾擊。

## 2. 日本被斷油決定跟美國拼了

- 日本在中國戰場卡住，又想佔領東南亞搶資源，結果惹毛美國。
- 美國立刻禁運，最致命的是切斷石油供應。
- 對日本來說，沒油就等於不能打仗，只剩兩條路：要麼認輸撤軍，要麼直接開打。
- 當時日本軍國主義高漲，加上過去「賭國運」常贏的迷信，還有海軍想壓過陸軍，最後決定孤注一擲，先炸癱美國海軍，好在談判桌上占便宜。

## 3. 炸掉珍珠港到底是贏還是輸？

- 1941年12月，日本偷襲珍珠港，出動幾百架飛機炸沉多艘美軍戰艦、摧毀上百架飛機。
- 軍事上這是一次成功奇襲，但戰略上卻是大敗。
- 尷尬的是，美國真正的主力——航空母艦——那天剛好都不在港內，讓日本錯失殲滅美軍海軍的機會。
- 更糟的是，這次偷襲給了美國一個完美理由，讓原本只想看戲的美國民眾，瞬間團結起來支持參戰。

## 4. 沉睡的巨獸醒來準備大復仇

- 珍珠港事件像一記重拳，徹底打碎美國的「不干涉主義」。
- 老百姓本來不想出國打仗，但自家軍港被炸後，全國怒火中燒，大家搶著報名當兵。
- 國會宣戰投票時，幾百人裡只有一位女議員反對。
- 隨著美國這個全球第一工業強國正式參戰，盟軍獲得源源不絕的武器和物資。
- 無論是中國、英國還是蘇聯，士氣都大振；而德國和日本，也即將親身體驗什麼叫「實力碾壓」。
