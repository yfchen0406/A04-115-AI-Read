# 二十一：美國打贏墨西哥搶大片地　舊金山掀起淘金熱

- 這段歷史主要講的是：19世紀中期，美國透過美墨戰爭一口氣擴張了國土，後來又因為加州淘金熱和摩門教西遷，引來大批人湧向西部。
- 但新土地到底要不要允許奴隸制度，讓南北方吵得不可開交，最後靠政治妥協才勉強拖住內戰爆發。

## 1. 痛扁墨西哥，美國地圖大升級

- 美國跟墨西哥開打，實力差太多，三路進攻輕鬆打到墨西哥首都。
- 最後墨西哥只能割地求和，把現在的加州、內華達、亞利桑那等地都讓給美國。
- 有趣的是，美國雖然打贏了，還「付錢」給墨西哥一千八百多萬美元當補償。
- 這場仗讓美國本土版圖差不多定型，也留下一堆西班牙語地名和大量墨西哥裔居民。

## 2. 新土地惹出大麻煩：到底能不能有奴隸？

- 領土變大後，最頭痛的問題就是：新州該不該允許蓄奴？
- 北方越來越覺得奴隸制很不道德，南方卻怕失去國會裡的優勢。
- 因為每一個新州加入聯邦，如果是「蓄奴州」或「自由州」，都會改變南北在國會的力量平衡。
- 這讓全國火藥味越來越濃，連林肯都開始公開反對奴隸制度。

## 3. 加州挖到黃金，全美國瘋狂西進

- 1849年，加州發現黃金，立刻引爆全球「淘金熱」，連清朝人都跑來碰運氣，還把這裡叫「舊金山」。
- 淘金的人怕奴隸來搶飯碗，所以加州傾向當「自由州」。
- 這下南方急了，甚至威脅說：「如果加州不准蓄奴，我們就退出美國！」
- 這場黃金夢，反而加速了南北衝突。

## 4. 老好人出面調停，內戰再拖十年

- 眼看國家快要分裂，資深政治家亨利·克萊提出「1850年妥協案」：讓加州以自由州加入，但北方要幫南方抓回逃走的奴隸，還給南方更多自治權。
- 這個方案暫時穩住了局勢，把內戰推遲了十年。
- 但大家心裡都清楚：這只是表面和平，真正的風暴遲早來。

## 5. 為信仰遠走他鄉，摩門教徒落腳鹽湖城

- 除了淘金客，還有一群摩門教徒為了躲避迫害，一路往西遷移。
- 他們的創始人史密斯自稱得到天使啟示寫了《摩門經》，卻在暴動中被殺。
- 接班人楊百翰帶領信徒歷經艱苦，最後在荒涼的鹽湖城定居下來。
- 雖然他們的一夫多妻等習俗被當時社會視為異端，但這群人的西遷，也成了開發美國西部的重要力量。

# 二十二：排外政黨崛起　堪薩斯點燃內戰導火線

- 這段講的是1850年代內戰爆發前的混亂局面：大量移民湧入引發本土主義浪潮，女權運動也在反奴隸運動中萌芽，而最關鍵的是「堪薩斯法案」打破了南北平衡，導致舊政黨瓦解、新黨（共和黨）誕生。

## 1. 外國人變多，有錢人搬去郊區

- 1850年前後，愛爾蘭和德國人大批移民美國。
- 愛爾蘭人因饑荒逃難，初來乍到很窮，只能擠在城市貧民窟，治安變差。
- 原本住在城裡的英裔有錢人紛紛搬到郊區避開。
- 這種「富人住鄉下、窮人住市區」的現象，反映了當時的族群與階級變化。

## 2. 神秘的「一無所知黨」

- 面對移民潮，本地人怕工作被搶，成立了一個叫「一無所知黨」的秘密政黨。
- 成員被問到黨內事，一律回答「我不知道」，因此得名。
- 他們靠排外情緒一度很紅，甚至差點選上總統。
- 但因為完全避談奴隸問題，內部意見不合，最後就解散了。

## 3. 幫黑人爭權利，才發現自己也沒投票權

- 很多女性積極參與反奴隸運動，結果發現自己連投票權都沒有，地位跟黑人差不多。
- 1848年，女權領袖斯坦頓和安東尼召開大會，模仿《獨立宣言》發表《情感宣言》，大聲要求男女平等。
- 雖然運動熱烈，但當時全國焦點還是集中在奴隸制度的爭議上。

## 4. 為了炒地皮，議員引爆火藥桶

- 堪薩斯想加入美國，卻卡在「要不要蓄奴」。
- 議員道格拉斯為了讓鐵路經過自己的地產、拉高房價，竟提出「人民主權」——讓當地人自己投票決定。
- 這招直接廢掉過去的妥協協議，導致堪薩斯爆發武裝衝突（史稱「流血堪薩斯」），也讓輝格黨垮台，催生了反奴隸的共和黨，林肯也因此登上政治舞台。

# 二十三：林肯辯贏道格拉斯　斯科特案惹怒全國

- 這段聚焦在1850年代內戰前夕的政治風暴，重點包括林肯與道格拉斯的大辯論、林肯的個人奮鬥故事，以及最高法院做出極具爭議的「斯科特案」判決，讓南北矛盾徹底失控。

## 1. 兩大政治明星公開對決

- 1858年，林肯和道格拉斯為爭奪參議員席位，在伊利諾州打了七場公開辯論。
- 道格拉斯主張「讓地方自己決定要不要奴隸」，林肯則堅決反對奴隸制度擴張。
- 雖然林肯輸了選舉，但他把辯論內容出版成書，一夜爆紅，為日後當總統鋪路。

## 2. 從底層爬起來的摔角高手——林肯

- 林肯童年很苦，家境貧寒、母親早逝，跟父親關係也不好。
- 他做過雜貨店員、自學法律，後來才進入政壇。
- 私生活也很坎坷，四個孩子死了三個。
- 有趣的是，他年輕時是摔角高手，曾連贏比賽後挑釁全場：「你們一起上啊！」
- 這些挫折反而鍛鍊出他的韌性。

## 3. 法院判黑人不是人，全國氣炸

- 當時有名奴隸斯科特，因為曾在自由州住過，告上法院要求自由。
- 結果最高法院不但判他敗訴，還宣稱「黑人不是公民，只是財產」，甚至宣布過去限制奴隸的法律無效。
- 這份判決站在歷史錯誤的一方，讓北方人怒不可遏，成為引爆內戰的關鍵導火線。

## 4. 大戰前最後的寧靜

- 到了1860年總統大選前，美國已瀕臨分裂邊緣。
- 林肯等人痛批斯科特案是「邪惡判決」。
- 全國上下劍拔弩張，只等一場大選來決定：美國是要繼續分裂，還是走向戰爭？
- 歷史的轉折點，就在眼前。

# 二十四：林肯當選總統　南方集體退出美國

- 這段講的是1860年內戰爆發的直接原因：從《湯姆叔叔的小屋》到約翰·布朗起義，南北對立不斷升溫；林肯當選後，南方各州乾脆宣布獨立；最終雙方開打，目標不是廢奴，而是「保衛聯邦統一」。：

## 1. 一本小說和一個狂人，點燃南北怒火

- 小說《湯姆叔叔的小屋》把黑奴的悲慘寫得太真實，讓北方人開始反思奴隸制，南方人則氣得寫書反駁。
- 後來激進分子約翰·布朗帶人攻打軍火庫，失敗後被處死。
- 北方把他當英雄，南方卻認為這是北方要動手的信號，雙方信任徹底破裂。

## 2. 林肯當選，南方立刻鬧分家

- 1860年大選，林肯在南方幾乎零票，但靠著北方人口優勢當選總統。
- 南方各州認為他一定會廢奴，乾脆在他還沒上任前，由南卡羅來納州率先宣布退出聯邦。
- 接下來幾個月，其他南方州也跟進，自己組成「美利堅邦聯國」。

## 3. 喊話沒用，南方直接開火

- 林肯在就職演說中溫情喊話：「我們是朋友，不是敵人。」
- 希望挽回統一。
- 但南方根本不理。
- 1861年4月，南方軍隊炮轟聯邦要塞，內戰正式開打。
- 其實林肯打仗的首要目標不是解放黑奴，而是「保住美國不分裂」。

## 4. 北方有錢，南方有猛將

- 開戰前雙方都在盤算：北方人多、工廠多、鐵路多、GDP高，像個富二代；南方雖然窮，但民風彪悍，還有一堆打過美墨戰爭的厲害將軍。
- 而且首都華盛頓離南方超近，這場仗注定不好打——果然打了整整四年。

## 5. 外國看戲，憲法模糊

- 英國、法國本來想趁機削弱美國，但因為支持奴隸制太不道德，不敢公開挺南方。
- 至於「州有沒有權退出聯邦」？
- 美國憲法根本沒寫清楚。
- 這場仗如果南方贏了，歷史可能完全不一樣。

# 二十五：林肯解放黑奴　葛底斯堡血戰定勝負

- 這段描述南北戰爭如何走向終結：從北方「蟒蛇封鎖」戰略，到林肯發布《解放宣言》提升戰爭意義，再到葛底斯堡血戰與經典演講，最後南方將軍光榮投降，美國重歸統一。

## 1. 用海軍像蟒蛇一樣勒死南方

- 戰爭初期，北方制定「蟒蛇計劃」：用海軍封鎖南方海岸，再拿下密西西比河，把南方切成兩半慢慢耗死。
- 但林肯一開始不敢提廢奴，只說「為統一而戰」，怕嚇跑邊境蓄奴州。
- 結果陸戰屢戰屢敗，吃盡苦頭。

## 2. 南方有戰神，北方換將像換衣服

- 南方的李將軍簡直是軍事天才，北方將軍換了一個又一個都打不贏。
- 有位叫胡克的將軍治軍太爛，軍營裡竟然出現大量妓女，據說英文「hooker」（妓女）就是從他名字來的。
- 儘管北方人多，但正面硬剛就是打不過李將軍，林肯壓力超大。

## 3. 《解放宣言》讓戰爭變「聖戰」

- 直到北方在安提耶坦戰役慘勝，林肯才敢發布《解放宣言》。
- 雖然實際解放效果有限，但它把戰爭昇華成「為自由而戰」的正義之戰。
- 這招讓英法找不到理由支持南方，北方成功掌握道德高地，南方徹底被國際孤立。

## 4. 最血腥戰役＋最短經典演講

- 葛底斯堡戰役打了三天，死傷慘重，是整場內戰最血腥的一仗。
- 戰後林肯在公墓發表不到300字的演講，強調「人人生而平等」，並喊出「民有、民治、民享」的名言。
- 這篇演講不僅悼念死者，更給美國一個重生的機會。

## 5. 放下武器，大家還是同胞

- 北方將軍格蘭特切斷南方補給，攻下里士滿（南方首都）。
- 李將軍眼看大勢已去，拒絕打游擊戰，選擇光榮投降。
- 格蘭特也很大氣，禁止士兵放炮慶祝，還提醒大家：「他們不是叛徒，是我們的同胞。」
- 這場美國史上最慘烈的內戰，就在互相敬禮中畫下句點。
